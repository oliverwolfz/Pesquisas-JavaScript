- É uma sintaxe introduzida no JavaScript (ES8) que simplifica significativamente a escrita e a leitura de código assíncrono. Ele permite que você escreva código que parece síncrono, mas que na verdade é executado de forma assíncrona, utilizando Promises por baixo dos panos.


- Declaradas com a palavra-chave async antes da palavra function. A palavra-chave await é usada dentro de uma função assíncrona.


- Pausa a execução da função até que a Promise seja resolvida.
