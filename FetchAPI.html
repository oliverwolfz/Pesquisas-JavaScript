- É uma interface JavaScript moderna e poderosa que fornece uma maneira simples e consistente de fazer requisições HTTP para servidores e obter dados. Ela é uma alternativa mais moderna ao antigo XMLHttpRequest.


- O método principal da Fetch API é o fetch(). Ele recebe uma URL como argumento e retorna uma Promise. Essa Promise representa a resposta da requisição.


- A Fetch API permite buscar recursos em toda a rede.
